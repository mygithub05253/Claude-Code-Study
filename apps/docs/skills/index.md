# 스킬 해설

Claude Code의 로컬 스킬들을 한국어로 해설합니다. 현재는 MVP(P0)의 10개 핵심 스킬이
공개되어 있으며, Phase 5(P1)에서 나머지 38개 스킬이 추가됩니다.

각 해설은 다음 6개 섹션을 포함합니다.

1. **한 줄 요약** — 이 스킬이 뭘 하는가
2. **언제 사용하나요?** — 사용 상황과 반대 상황
3. **핵심 개념** — 원칙과 철학
4. **실전 예제 (대학생 관점)** — Next.js 15 + 동아리 공지 게시판
5. **학습 포인트** — 공부하는 입장에서의 팁
6. **원본과의 차이** — 원본 SKILL.md 대비 각색 지점
