# My Collection — 커스텀 리소스

> 원본 리소스를 **변형·확장해서** 만든 사용자 전용 Skills / MCP 설정 / Hooks / CLAUDE.md 템플릿 / 프롬프트를 모읍니다.

## 이 섹션의 목적

- 원본을 그대로 쓰지 않고 한국어 대학생 맥락에 맞게 변형한 결과물을 공유합니다.
- 모든 커스텀 리소스는 **원본을 명시** 하고 변경점을 한국어로 기록합니다.

## 하위 분류

| 하위 카테고리 | 내용 |
|---|---|
| 🔧 커스텀 Skills | 원본 Skill + 한국어 프롬프트 변형 |
| 🔌 MCP 설정 | 프로젝트 유형별 `settings.json` 스니펫 |
| 🪝 Hooks | 개인 워크플로우 자동화 |
| 📄 CLAUDE.md 템플릿 | 프로젝트 유형별 템플릿 |
| 📝 프롬프트 모음 | 검증된 재사용 프롬프트 |

## 기여 규칙

1. **원본 반드시 명시** — Frontmatter `source_url` 필수
2. **변경 사유 기록** — "왜 변형했는가" 를 본문 상단에 명시
3. **라이선스 호환성 확인** — 원본 라이선스가 재배포를 허용하는지 확인 후 등록
