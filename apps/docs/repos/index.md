# GitHub 인기 레포 큐레이션

> Stars 10K+ 레포지토리 중 **Claude Code와 시너지가 좋은** 프로젝트를 한국 대학생 관점에서 선별합니다.

## 이 섹션의 목적

- 학부 과제·졸업작품·공모전·포트폴리오에 바로 쓸 수 있는 레포를 카테고리별로 소개합니다.
- 각 레포를 Claude Code에서 `clone → 이해 → 변형` 하는 워크플로우를 제시합니다.
- 원본 코드는 복제하지 않고 링크와 한국어 요약만 수록합니다.

## 카테고리

| 카테고리 | 예시 | 대학생 활용 맥락 |
|---|---|---|
| 프론트엔드 프레임워크 | Next.js, React, Vue | 과제/프로젝트 |
| 백엔드 & API | FastAPI, Express, NestJS | 졸업작품, 공모전 |
| AI & ML | LangChain, Transformers | 캡스톤, 연구 |
| DevOps & 배포 | Docker, Vercel | 포트폴리오 배포 |
| 문서 & 자동화 | python-docx, Pandoc | 보고서, 논문 |
| UI 라이브러리 | shadcn/ui, Tailwind | 디자인 시스템 |
| 데이터 분석 | Pandas, D3.js | 데이터 과목 |
| 보안 & 테스트 | OWASP, Jest | 품질 관리 |

## 기여 가이드

1. `content/ko/repos/[repo-name].md` 생성
2. Frontmatter `source_url` 은 GitHub 원본 URL, `license` 는 정확한 라이선스 ID(MIT/Apache-2.0/등)
3. 6섹션 템플릿 준수, 실전 예제는 "동아리 공지 게시판" 맥락으로 통일
