# 대학생 실무 카테고리

> Skills / MCP / Hooks / Agents / Repos 를 **실제 대학생 업무 상황별** 로 묶어 소개합니다.

## 카테고리

| 카테고리 | 다루는 내용 |
|---|---|
| 📝 문서화 & 보고서 | 과제, 논문, 발표자료, Markdown → PDF |
| 💻 코딩 & 디버깅 | 프로그래밍 과제, 코드 리뷰, 리팩터링 |
| 📊 데이터 분석 | 통계, 시각화, 크롤링, CSV 처리 |
| 🏗️ 프로젝트 관리 | 팀 프로젝트, 스프린트, TODO 정리 |
| 🎨 UI/UX 디자인 | 프론트엔드, 프로토타입, shadcn/ui |
| 🚀 배포 & DevOps | CI/CD, GitHub Pages, Vercel |
| 📋 포트폴리오 & 취업 | 이력서, 기술 면접, README 작성 |
| 🤝 팀 협업 | Git, 코드 리뷰, 커뮤니케이션 |

## 향후 작성 방향 (Phase 5)

각 카테고리별로 **"시나리오 → 추천 리소스(Skill/MCP/Hook/Repo) → 프롬프트 예시 → 실전 워크플로우"** 순서로 한 페이지씩 작성합니다.
