# MCP 서버 해설

> **MCP(Model Context Protocol)** 는 Claude Code에 외부 도구·데이터 소스를 연결하는 표준 프로토콜입니다. 이 섹션은 대학생이 자주 쓰는 MCP 서버를 한국어로 정리합니다.

## 이 섹션의 목적

- 공식·커뮤니티 MCP 서버 중 **대학생 프로젝트에 유용한 것**만 선별해 소개합니다.
- 설치 단계, `settings.json` 예시, 실전 활용 시나리오를 한국어로 제공합니다.
- 원본 코드는 복제하지 않고 링크와 요약만 수록합니다.

## 향후 큐레이션 대상 (Phase 4)

| MCP 서버 | 기능 | 대학생 활용 |
|---|---|---|
| Sequential Thinking | 단계별 추론 | 알고리즘 설계, 아키텍처 결정 |
| GitHub MCP | Git 연동 | PR 관리, 이슈 트래킹 |
| Filesystem MCP | 파일 시스템 | 로컬 프로젝트 관리 |
| Fetch MCP | 웹 크롤링 | 리서치, 자료 수집 |
| Supabase MCP | BaaS | 풀스택 프로젝트 |

## 기여 가이드

1. `content/ko/mcp/[server-name].md` 파일 생성
2. 프로젝트 표준 6섹션 구조 준수 (요약 / 시나리오 / 핵심 개념 / 실전 예제 / 학습 포인트 / 원본 링크)
3. Frontmatter에 `source_url`, `source_author`, `license`, `last_reviewed` 필수
